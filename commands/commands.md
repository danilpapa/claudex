# Commands

`/clear` - clear current context (usefull after finishing a task)
`/rewind` - rollback the last message
`/compact` - ai summarizes current thread and clears the context
`/compact <rule>` - summarizin ONLY highlighted information
    - /compact keep track the current architecture and the latest changes in network services
`@file_name` - highlight the file
`claude --continue` - resume the closed chat
`claude mcp list` - list available MCP's

# Hot keys
`ctrl+v` - past the copyed image
`tab` - switch "thinking" mode OR use 'think'/'think harder' in the prompt
